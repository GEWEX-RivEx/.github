# Welcome to GEWEX-RivEx 🌊

Welcome to the official GitHub organization for the **GEWEX River Experiment Initiative (RivEx)**. We are a community dedicated to advancing global hydroclimatology and river routing research.

This space serves as the central hub for our open-source tools, collaborative discussions, and community resources.
